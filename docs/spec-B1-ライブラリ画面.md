## 0. 原則・共通ルール

- **ナビ**：5タブ（ホーム / カレンダー / ＋記録 / **ライブラリ** / マイページ）。ライブラリ上部に**カテゴリタブ**（すべて＋各柱・カテゴリ色）。
- **タブ切替＝別ビュー**：各ジャンルは「**既定ビュー＋見方切替（サブタブ）**」を持つ。汎用の一覧を全ジャンルで使い回さない。
- **「すべて」タブ**：比率が混ざるので**タイムライン/フィード**（横断・日付順）で受ける。
- **★per-genreカレンダーは作らない（2026-07-06 確定）**：日付で見る＝**グローバルの「カレンダー」タブ**（カテゴリのレイヤーON/OFFで各ジャンルだけ表示可）に一本化。各ジャンルのライブラリは**ギャラリー/グリッドに専念**（重複を避けシンプルに）。
- **並び替え・絞り込み・検索**：**各ジャンルの具体は後回し（2026-07-06 ユーザー指示）**。まず「既定ビュー＋見方切替」を各ジャンルで固め、フィルタ/ソートは後で一括設計する。
- **リピート**：ジャンルの流儀で（映画=日記に複数、酒=味覚マップに点、書籍=シリーズ、施設=通算）。
- **★スコープ（2026-07-06 明確化）＝本書は"見せ方（一覧の視覚レイアウト）"だけ**。「この監督のn本／このキャストn回」等の**横断カウントは統計＝A5（別画面）**。見せ方には含めない（例：日記・ギャラリー・マップ・棚は"並べ方"なので見せ方、"n回"は統計）。
- **設計思想（★2026-07-06 明確化）＝各ジャンルは専用アプリを"超える"のが目標**：映画がFilmarks/Letterboxdを超えたのと同じ公式を全ジャンルに適用＝**①見やすさを借りる ②専用アプリの弱点を突く ③横断（雑食）＋私的・美しさで上回る**。真似でなく超える。
  - 例外的に避けること：**DIYツール（Trello等）っぽいカンバンにしない**。状態管理も"記録の中に美しく溶かす"（下記 観劇のチケット状態）。
- **★ジャンルごとの色世界（2026-07-06 確定）**：**共通素材＝Liquid Glass／全体の基調＝白**。ただし**各ジャンルのライブラリは、そのジャンルのカテゴリ色を"世界観"としてまとえる**——映画＝**ダーク基調のガラス**（ポスターが映える）、観劇＝**ワインレッド＋ゴールド**（緞帳の赤・装飾の金＝劇場の世界観）、酒＝琥珀基調、御朱印＝朱基調…。白ベースは崩さないが、ジャンルタブに入ると色がつく＝雑食でも各柱に個性。エキスパート表示（レイアウト差）＋色世界（トーン差）の二層で差別化。

## 1. 映画【確定 2026-07-06】

**キャッチ：Filmarksの見やすさ＋Letterboxdの監督横断＋favorecoの私的・美しさ**

- **色世界＝クリーンな白ベース（2026-07-06 再確定）**：ダークキャンバス（背後薄暗）は"いまいち"で**不採用**。骨格も面も白/ガラス＝Filmarks的なクリーンさ。ポスターの色だけで十分映える。レイアウト＝**3列ポスター**。
- **構造＝1画面で「見たい(Interest)／見た(Watched)」に分割（Mystorium流・2026-07-06 確定）**：タブ切替でなく、**1画面に見たい（上・公開予定バッジ）と見た（下・★評価）の2セクションを縦に並べる**。**Journal（日記/時系列）は別モード**（上部で ライブラリ⇄Journal 切替）。
  - **見たい（Interest）**＝まだ観てない映画（記録の前段階・書籍の積読と同型）。TMDb検索で追加、未公開作は**公開日（日本）も取得**し「公開予定 7/10」バッジ。公開日は Coming Up・カレンダーに"予定"として流れ込む（チケット発売と同型）。観たら鑑賞日/評価を足して「見た」に昇格（Eventはそのまま・Visitが付く）。**自分の見たいリストの公開日だけ＝私的**（Trend browseは作らない・§要検討）。
- **ポスター下ステータスアイコン（2026-07-06 ユーザー要望）**：各ポスターの下に**自分のステータス**を小さく表示＆タップで切替＝**👁見た（緑）／＋興味あり（青）／★評価（見た時）**。Filmarksは"他人の数字"だが、favorecoは**自分の状態**（私的）。どのポスターからでも状態を一目＆その場で変えられる。
- **評価＝★5段階0.5刻み**（RESONANCE・全ジャンル統一・§9）。Journal（日記）は不採用（カレンダーと重複・§9）。
- **★"見たい/見た"の分割は汎用パターン**：Mystorium流の Interest/Done を1画面で分ける形は、書籍（積読/読了）・御朱印（気になる/参拝済）など各ジャンルに横展開できる。
- **見せ方（視覚レイアウト）＝ポスターウォール（密・コレクション一望）**：
  - **既定＝密なポスターウォール（2:3・3〜4列で詰める・chrome最小・ポスターが語る）**。本数が多く"一望する台帳"の文化（Letterboxd Films / Filmarks流）。
  - **日記（Journal/Diary）**＝月ごとに鑑賞を時系列で並べる（再鑑賞も見える）＝Letterboxd Journal流。
  - タブ構成の参照：Letterboxdの **Films / Journal**（Reviews/Lists/friendsのソーシャルは私的方針で除外）。ポスター下に小さな★評価。
- **佇まい**：ダーク・密・小さめタイル・コレクション感。
- **参照**：Letterboxd（ダーク・密ポスターウォール・Journal）／Filmarks（ポスターグリッド）。
- ※「この監督/俳優 n本」の横断は**統計＝A5**（別画面）。フィルタ/並び替えも後回し。

## 2. 観劇【表示方向 確定 2026-07-06／フィルタは後回し】

**ゴール：シアティ（Theaty）を超える**（真似でなく上回る）。

- **色世界＝ワインレッド＋ゴールド（2026-07-06 確定）**：緞帳の赤・装飾の金＝劇場の世界観。映画のダークと好対照。Liquid Glass素材にワインレッド／ゴールドのトーン。
- **見せ方（視覚レイアウト）★2026-07-06 確定（カレンダー切替は廃止）**：
  - **ライブラリ＝ギャラリー（チラシ）のみ**。「いつ観たか」で見たいときは**グローバルの「カレンダー」タブ（観劇レイヤーON）**に任せる。**per-genreカレンダーは重複＝廃止**（映画と同じ構造：ライブラリ＝グリッド／日付＝カレンダータブに一本化）。★2026-07-06 ユーザー指摘で修正。
  - **観たい / 観た 分割**（Mystorium流）＋**チラシ下ステータスアイコン**（🔖観たい / 👁観た / ★評価）＋**観たいにチケット状態バッジ**（当落待ち等）。
  - **チケット要対応ストリップ**（当落待ち/入金待ち/発券待ち）を上部に常時表示（記録に溶かす・Trelloカンバンにしない）。
  - **密度＝3列（既定・Theaty風の詰め）／2列（切替オプション）**。**上質さは密度でなく色世界（ワインレッド＋ゴールド・クリーム地＋金枠チラシ）と質感で出す**。チラシ＝A4縦。
- **佇まい**：詰め気味の3列＋ワインレッド/ゴールドの劇場感（全画面ダークにはせず、共通骨格＝クリーム地に金枠チラシ）。
- **チケット状態＝"美しく統合"（Trello自作カンバンにしない）**：
  - 上部に**「要対応」ストリップ**：`📩当落待ち3` `🔴入金待ち2` `🟡発券待ち1`（期限つき・タップで該当公演へ）＝一目で"今やること"（※これは"見せ方"上の機能表示。件数は状態の数であって集計統計ではない）
  - 各公演のチラシに**小さな状態バッジ**（色ドット）
  - → 記録（チラシウォール）と状態が**1画面に溶ける**。Trelloの分断も、Theatyの"状態管理なし"も超える
- **超えどころ**：チケット状態の統合／横断（雑食）／チラシの美しさ。
- **参照**：シアティ（座席・感想無制限・思い出コレクション・カレンダーで観劇ログ／単一ジャンル・状態管理なし）。
- ※キャストで束ねる横断は**統計＝A5**。フィルタ/並び替えは後回し。

## 5. 酒【表示方向 確定 2026-07-07／フィルタは後回し】

> ゴール：さけのわ/Untappd の集計は借りつつ（読み取りAPI）、**"自分のボトル棚"を美しく並べる私的ライブラリ**で見せる。デザインの磨きはデザインフェーズ（モックのCSSボトルは仮・実際は登録写真）。構造・振る舞いを本節で確定。

- **色世界＝琥珀（アンバー×蜂蜜）**。
- **★カードの構造＝〈名前バンド〉＋〈全体写真（ボトル）〉＋〈ラベル表・裏〉＋〈下部に情報〉**（ユーザースケッチ 2026-07-07）。
  - 名前バンド：銘柄＋蔵/種別。
  - 全体写真（ボトル）が主役。銘柄ごとにボトルの形・色が違うのが並べたときの魅力。
  - **ラベル表・裏の2枠**。**裏ラベルが無い場合は右枠を味わいレーダー（さけのわ6軸）に自動差し替え**。
  - 下部情報：酒蔵・種別・精米歩合・日本酒度/酸度（クエリ可能な個別列＝A3 §5）。
- **★密度3段（表示サイズ切替）**：
  - **最小＝3列アイコン**（名前バンド＋ボトルのみ＝スケッチの最小形）。
  - **コンパクト＝2列**（＋産地・★評価）。既定はここ。
  - **フル＝1列**（＋ラベル・スペック・レーダー）。
  - 同一の「銘柄カード」が密度で伸縮。眺める時は最小、じっくり見る時はフル。
- **タップで詳細**：スペック個別列（精米歩合/日本酒度/酸度/アルコール）＋**さけのわ6軸レーダー × 味覚マップ2軸（甘辛×濃淡・自分の点）**＋官能メモ（香→味→余韻）＋評価＋**酒蔵ハブへの横断リンク**（「この蔵の記録n件」・A6）。
- **酒キラーの見せ方（A5と連動）**：**酒蔵所在県マップ**（どこの酒が好きか）／**味覚マップ**（好みの中心）。
- **参照**：さけのわ（日本酒版Untappd＝集計は借りる・SNSは作らない・02勝ち筋#4）／Untappd（チェックイン文化）。favorecoは"自分のボトル棚＋味覚マップ＋蔵ハブ"で私的体験に振る。
- ※種別（純米大吟醸/純米/生酛…）・産地はフィルタ・後回し。統計（種別内訳・県別・味覚集計）はA5。

## 9. 共通の見せ方機能（全ジャンル・2026-07-06 確定）

Letterboxdの売りから採用した、ジャンルをまたぐ見せ方：

- **Lists（マイベスト・自作リスト）**：ユーザーが**順位つき/テーマ別のリスト**を作れる（「マイベスト観劇10」「今年の本」等）。ジャンル内でも横断でも。眺めて嬉しい＋好みが結晶する。※ソーシャル共有はしない（私的）。
- **推しピン（Four Favorites）**：各ジャンル（や全体）の**推しを大きく固定表示**＝favorecoの「柱」思想の顔。プロフィール/カテゴリ上部に。
- **ヒートマップ（体験頻度）→ 年間まとめ(A5)**：GitHub草グラフ的に体験の頻度を色濃淡で。年間まとめ（買い切り）の顔に。
- **評価＝★5段階0.5刻み（RESONANCE）で全ジャンル統一**（映画も）。100点/10点は消費者標準でなく迷いやすいため不採用（映画だけ"100点詳細モード"を望むならオプション余地・既定は★）。※Letterboxd/映画.com/Filmarksも5段階系。
- **Diary（時系列日記）は不採用**（カレンダーと重複・シンプル優先。2026-07-06）。
