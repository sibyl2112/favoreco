## 0. 原則・共通ルール

- **ナビ**：5タブ（ホーム / カレンダー / ＋記録 / **ライブラリ** / マイページ）。ライブラリ上部に**カテゴリタブ**（すべて＋各柱・カテゴリ色）。
- **タブ切替＝別ビュー**：各ジャンルは「**既定ビュー＋見方切替（サブタブ）**」を持つ。汎用の一覧を全ジャンルで使い回さない。
- **「すべて」タブ**：比率が混ざるので**タイムライン/フィード**（横断・日付順）で受ける。
- **並び替え・絞り込み・検索**：**各ジャンルの具体は後回し（2026-07-06 ユーザー指示）**。まず「既定ビュー＋見方切替」を各ジャンルで固め、フィルタ/ソートは後で一括設計する。
- **リピート**：ジャンルの流儀で（映画=日記に複数、酒=味覚マップに点、書籍=シリーズ、施設=通算）。
- **設計思想（★2026-07-06 明確化）＝各ジャンルは専用アプリを"超える"のが目標**：映画がFilmarks/Letterboxdを超えたのと同じ公式を全ジャンルに適用＝**①見やすさを借りる ②専用アプリの弱点を突く ③横断（雑食）＋私的・美しさで上回る**。真似でなく超える。
  - 例外的に避けること：**DIYツール（Trello等）っぽいカンバンにしない**。状態管理も"記録の中に美しく溶かす"（下記 観劇のチケット状態）。

## 1. 映画【確定 2026-07-06】

**キャッチ：Filmarksの見やすさ＋Letterboxdの監督横断＋favorecoの私的・美しさ**

- **既定ビュー＝ポスターウォール（2:3・3列）**
- **見方切替（サブタブ）**：作品 / **監督** / **俳優** / ジャンル / **日記（Diary）**
  - 監督・俳優＝人物で束ね「この監督/俳優 n本」＝**フィルモグラフィ（自分の鑑賞史）**。★**Filmarksに無い差別化**（A5 §6）。
  - 日記＝月ごとに鑑賞を並べる（再鑑賞も見える）＝Letterboxd Diary流。
- **並び替え**：鑑賞日 / 評価 / 公開年 / タイトル（Filmarks準拠）
- **絞り込み**：ジャンル / 形態（劇場/IMAX/4DX/配信）/ 評価
- **favorecoらしさ**：私的・非ソーシャル（Letterboxdは社交、うちは自分のため）／舞台挨拶・応援上映はタグ→「今年n回」／リピート＝日記に複数エントリ
- **参照**：Filmarks（並び替え=投稿日/評価/製作年/タイトル/鑑賞日・監督横断は不可）／Letterboxd（Diary・監督/俳優の専用ページ・多彩なsort/filter）

## 2. 観劇【表示方向 確定 2026-07-06／フィルタは後回し】

**ゴール：シアティ（Theaty）を超える**（真似でなく上回る）。

- **既定ビュー＝チラシ/ポスターウォール（A4縦）**（映画に近いが縦長比率）
- **見方切替（サブタブ）**：
  - 公演（チラシグリッド）
  - **出演者（キャスト）横断**（人物で束ね「この人を n回観た／出演公演を串刺し」）★**Theatyに無い＝超えどころ**。映画の監督横断と同じ武器
  - 日記 / カレンダー（観劇ログを日付で見返す）
- **チケット状態＝"美しく統合"（Trello自作カンバンにしない・要修正点だった）**：
  - 上部に**「要対応」ストリップ**：`📩当落待ち3` `🔴入金待ち2` `🟡発券待ち1`（期限つき・タップで該当公演へ）＝一目で"今やること"
  - 各公演のチラシに**小さな状態バッジ**（色ドット）
  - → 記録（チラシウォール）と状態が**1画面に溶ける**。Trelloの分断も、Theatyの"状態管理なし"も超える
- **超えどころまとめ**：キャスト横断／チケット状態の統合／横断（雑食）／チラシの美しさ。
- **参照**：シアティ（座席・感想無制限・思い出コレクション・カレンダーで観劇ログ／単一ジャンル・状態管理なし）／Trello自作（状態ボード＝分断・DIY）。
- フィルタ/並び替えは後回し。
