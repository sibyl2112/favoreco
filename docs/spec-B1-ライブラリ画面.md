## 0. 原則・共通ルール

- **ナビ**：5タブ（ホーム / カレンダー / ＋記録 / **ライブラリ** / マイページ）。ライブラリ上部に**カテゴリタブ**（すべて＋各柱・カテゴリ色）。
- **タブ切替＝別ビュー**：各ジャンルは「**既定ビュー＋見方切替（サブタブ）**」を持つ。汎用の一覧を全ジャンルで使い回さない。
- **「すべて」タブ**：比率が混ざるので**タイムライン/フィード**（横断・日付順）で受ける。
- **並び替え・絞り込み・検索**：**各ジャンルの具体は後回し（2026-07-06 ユーザー指示）**。まず「既定ビュー＋見方切替」を各ジャンルで固め、フィルタ/ソートは後で一括設計する。
- **リピート**：ジャンルの流儀で（映画=日記に複数、酒=味覚マップに点、書籍=シリーズ、施設=通算）。
- **設計思想**：既存の"見やすさ"を借り、既存の"弱点"を突き、favorecoの"私的・美しさ"を足す。

## 1. 映画【確定 2026-07-06】

**キャッチ：Filmarksの見やすさ＋Letterboxdの監督横断＋favorecoの私的・美しさ**

- **既定ビュー＝ポスターウォール（2:3・3列）**
- **見方切替（サブタブ）**：作品 / **監督** / **俳優** / ジャンル / **日記（Diary）**
  - 監督・俳優＝人物で束ね「この監督/俳優 n本」＝**フィルモグラフィ（自分の鑑賞史）**。★**Filmarksに無い差別化**（A5 §6）。
  - 日記＝月ごとに鑑賞を並べる（再鑑賞も見える）＝Letterboxd Diary流。
- **並び替え**：鑑賞日 / 評価 / 公開年 / タイトル（Filmarks準拠）
- **絞り込み**：ジャンル / 形態（劇場/IMAX/4DX/配信）/ 評価
- **favorecoらしさ**：私的・非ソーシャル（Letterboxdは社交、うちは自分のため）／舞台挨拶・応援上映はタグ→「今年n回」／リピート＝日記に複数エントリ
- **参照**：Filmarks（並び替え=投稿日/評価/製作年/タイトル/鑑賞日・監督横断は不可）／Letterboxd（Diary・監督/俳優の専用ページ・多彩なsort/filter）
