# ②-A / spec-A4 テンプレ別プリセット具体リスト

> ステータス: 清書（2026-07-06）
> 位置づけ: ②-A のプリセット分冊。各テンプレの**評価軸名・成果選択肢・形態選択肢・作品ジャンル選択肢・スペックチップ・種別サブレイヤーの中身**を具体化。フィールドの器はA3、種別の仕組みはA2。
> 凡例: **【確定】**＝08のヒアリングで確定済み ／ **【提案】**＝08に明記が無く本書で埋めた案（要ユーザー確認）。すべて**ユーザー編集可**（プリセットは初期値）。
> 関連: A2（種別サブレイヤー）／ A3（フィールド）／ 08 §4-2〜§4-11

---

## 0. 全テンプレ共通の考え方

- プリセットは**初期値**。ユーザーは選択肢の追加/リネーム/削除ができる（OFFにしても入力済みデータは消さない・08 §4）。
- 選択肢マスター（成果/形態/ジャンル/評価軸）は**カテゴリ（＋種別）スコープ**で持つ（A2解決）。
- 成果・評価軸は「同じユニットを使うカテゴリで同じ統計が成立する」ためキーで管理。

## 1. 総合評価（RESONANCE型）

- 全カテゴリ共通の**単一総合評価**（Visit・A3土台）。Mystorium由来の呼称「RESONANCE」を踏襲。
- **【確定 2026-07-06】★0.5刻みの5段階（0.5〜5.0）**。Double(0.5刻み)で保存し統計・ソートに使う（Filmarksと同じ半刻み）。
- U7評価軸レーダーは総合評価とは別（軸別の詳細・オフ開始のテンプレが多い）。

### ★成果(U8)は全ジャンル廃止（2026-07-06 ユーザー判断）
「総合評価があれば十分・争うものでもない」。**U8成果を全テンプレから外す**（08マトリクスの●も無効）。ただし書籍は、成果ではなく **共通興味ステータス＋書籍固有の入手状態＋購入済み後の任意読書状態** として扱う（§9）。

- **キャスト取り込み精査**：役名×人物名のペアを精査画面で編集してから保存。画像は前処理・列解析・Person辞書補正、URL/テキストは候補抽出を行い、勝手に確定しない。保存先はEvent/Visit/両方から選ぶ。
- **コレクション(U11)●**：チラシ / パンフ / 半券 / グッズ等。写真＋名前/コメント＋タグ（アクスタ/チラシ/缶バッジ等）＋金額任意。写真から後でコレクション化できる。

## 6. 酒【確定＋日本酒プリセット】

### 酒種別サブレイヤー（A2）
- 種別：**日本酒 / ビール / ワイン / ウィスキー / 焼酎 / その他**（v1は下地のみ通し、中身は**日本酒プリセットだけ実装**）。表示タブは並び替え・表示/非表示をユーザー設定で変更可能。

### 全酒種共通ベース
- **Event共通**：名前 / 酒種別 / 度数 / 代表写真 / 気になる状態。
- **Visit共通**：飲んだ日 / どこで飲んだ / 評価 / メモ / 写真 / リピート回数集計。
- **Hero候補**：最近飲んだお酒 / お気に入りからランダム / 手動でトップに出す / 同じ季節に飲んだお酒。
- **一覧単位**：飲んだ一覧は銘柄(Event)単位。同じ酒は1カードにまとめ、再飲はVisitとして積む。
- **気になる→飲んだ**：気になる酒を飲んだら、Eventへ昇格しVisitを追加。メモ・写真・URLは引き継ぐ。
- **写真**：代表写真はボトル/グラス/ラベルのいずれでも可。ラベル表/裏はLv2詳細。
- **データ**：酒種別比率 / 今月の種類数 / 年間の種類数 / 地域 / 生産者・酒蔵 / 評価。選択中タブに応じて絞る。
- 酒種別ごとの固有情報はプリセットで足す。共通ベースへ日本酒専用項目を混ぜない。
- **詳細入力の段階開示**：
  - Lv1（既定）：名前 / 酒種別 / 写真 / 飲んだ日 / 場所 / 評価 / ひとことメモ。
  - Lv2（詳しく記録）：度数 / 生産者 / 地域 / 飲み方 / 価格 / 買った場所 / 同行者 / ラベル写真 / タグ。
  - Lv3（専門スペック）：酒種別プリセットの詳細値。初期は閉じる。

### 日本酒プリセット
- **味覚マップ2軸(U17)●**：横=甘辛度（辛口⇔甘口）／縦=濃淡度（淡麗⇔芳醇）。4タイプ表示＝**薫酒 / 爽酒 / 醇酒 / 熟酒**。さけのわ数値を参考デフォルト表示。
- **評価軸レーダー(U7)○（さけのわ6軸準拠・確定）**：華やか / 芳醇 / 重厚 / 穏やか / ドライ / 軽快（対の関係：華やか⇔穏やか・軽快⇔重厚）
- **官能メモ(U10)○**：香り → 味わい → 余韻
- **スペック表(U9)●チップ**：蔵元 / 銘柄名 / 使用米 / 酵母（＋自由追加）
- **昇格数値列（Event個別・検索/グラフ）**：精米歩合 / 日本酒度 / 酸度 / アルコール分
- ~~成果(U8)~~ 廃止（§1）。「リピート買いたい」はタグ or リピート機能で代替
- **OCR辞書(U16・裏ラベル定型)**：精米歩合 / 日本酒度 / 酸度 / アルコール分 / 使用米 / 酵母 / 製造年月
- 銘柄検索＝さけのわAPI／酒蔵マスター＝所在県マップ集計元

## 11. 持ち越し（A5/A6へ）

- **解決済み（2026-07-06〜08）**：RESONANCE=★0.5刻み5段階／成果=全廃／さけのわ6軸=確定／音楽・書籍ジャンル=調査反映／映画評価軸=脚本/演出/演技/映像/音楽（既定オフ・確定）／**美術展=総合評価のみ（確定）**／書籍は**共通興味ステータス＋入手状態＋任意読書状態**に分離（確定）／**種類↔ジャンル=緩い連動**／**コレクションにキャプション（任意）・写真→コレクション移動可**（A3）。
- 残る要確認：なし（A4は確定）。
- A5：選択肢を使った統計（ジャンル別・軸平均・味覚マップ集計・**読書状態別＝積読リスト**等）。
- A6：選択肢マスターのkey/表示名分離（A2同様・リネームで既存記録を壊さない）。
