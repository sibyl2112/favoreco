## 総論

1. **「〇〇印」制度が御朱印の外へ大増殖している**。御城印・鉄印・御船印（フェリー）・御翔印（空港）・御宿印（宿）・御酒印（酒蔵）など、専用の「印帳」商品が各制度で販売され、収集文化が制度化されている。個別ジャンルを追うより、**「印・スタンプ・カードを貼って残す」汎用フィールドを持てば多ジャンルを一気に拾える**。
2. **公共配布カードの世界が広い**。ダム・マンホールに加え、発電所・砂防・橋・トンネル・古墳・文化財・離島・工場夜景など自治体単位で制度が乱立。個人運営のまとめサイト（カードハンター等）はあるが**横断記録アプリは空白**。
3. **LiveNote（ライブ参戦記録・個人開発）が音楽以外に転用されている**（プロレス・クラシック・お笑い）。シアティも「舞台以外のコンサート・お笑いの記録用」に使われている。＝**「現場参戦」を横断する記録の需要は実在し、専用の王者は不在**。
4. **紙の専用ノートが売れているジャンルはデジタル空白の逆引きになる**。推し活手帳（インプレス・28演分のレポ欄）、ダイソー推し活ノート（9色展開）、伊東屋ホリックログ（趣味別リフィル）、ダイゴーの読書×コーヒー/ウイスキー/紅茶ノート、競馬予想の自作リフィル配布、万年筆インクの色見本帖文化など。

## favoreco設計への示唆

1. **「印・スタンプ・カード」を汎用フィールドに**: 御朱印帳型の記録（印の写真＋取得場所＋日付＋一言）は、御船印・御城印・カード類・スタンプまで同じ構造。カテゴリテンプレとは独立した「コレクション」フィールドとして持つと1機能で多ジャンルを吸収できる
2. **「現場参戦」は横断テンプレが正解**: 音楽ライブ・クラシック・落語・お笑い・プロレス・トークイベントは「会場・出演者・座席・セトリ/演目・同行者」でフィールドがほぼ共通。細分化せず1テンプレ＋カスタムが効率的
3. **テーマパークは第1弾テンプレ候補に昇格を検討**: 記録文化が濃く空白度が高く、写真主役・繰り返し訪問（リピートグループ資産が活きる）とfavorecoの構造に合う
4. **御朱印はテンプレ優先度を下げる**: 専用アプリが複数強い。「印」汎用フィールドで拾えれば十分
5. **紙ノートが売れているジャンル一覧（学研ステイフル・ダイゴー・伊東屋等の商品ライン）は今後のテンプレ追加のロードマップ**としてそのまま使える（ペット記録・レシピ・ドラマ視聴など）
