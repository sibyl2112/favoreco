# 競合分析：「好きなものを記録する」汎用記録アプリ

> 調査日: 2026-07-04（Mystorium プロジェクトのセッションで実施した Web リサーチの正本）
> 対象: 書籍 / 映画 / 美術館・博物館 / 演劇・観劇 / 食べ歩き・飲み歩き / 汎用記録 / 推し活・ライブ参戦

---

## 総論（先に結論）

- **書籍・映画は「巨大なソーシャルDB型無料アプリ」に完全に押さえられている**。ここで汎用アプリが記録先として選ばれるのは困難。読書メーター/ブクログ/Filmarks/Letterboxd はデータベース（書誌・作品DB）と他人のレビューという、個人開発では複製不可能な資産を持つ。
- **美術館・演劇・ライブ参戦・食べ歩き（自分用）は明確に手薄**。この領域の既存アプリは (a) 情報ポータルのおまけ機能（チラシミュージアム、シアティ）、(b) 個人開発の小規模アプリ（Punct Diary, TheaterRecords, LiveNote, LIVE DIARY）のどちらかで、「美しい・私的・統計が豊か・買い切り」を全部満たすものは見当たらない。
- **「なんでも記録」汎用枠には既に直接競合が存在する**（キログ、「記録｜読書も映画も」、海外では Sofa）。ただしいずれもデザイン性・写真体験・レーダーチャートのような「体験の質」では戦っておらず、機能表で戦っている。差別化余地はあるが「汎用＝誰の一番にもならない」リスクは現実的。
- **買い切りモデルは差別化として機能する**。Day One(~$50/年)・Sofa($30/年)・StoryGraph($5/月)などサブスク疲れへの不満は英語圏フォーラムで顕著。日本でもキログが「買い切り課金・控えめ広告」で好評を得ており、需要の証拠がある。ただし買い切りは LTV が低く、継続開発の資金源としては弱い——これは競合の弱点であると同時に自分の制約でもある。

---

## 1. 書籍

| アプリ | モデル | 規模感 |
|---|---|---|
| 読書メーター（dwango系） | 無料＋広告 | 国内最大級。レビュー1,000万件超（2014年時点） |
| ブクログ | 無料＋広告＋プレミアムサブスク | 登録会員200万人突破（2024年、20周年） |
| Goodreads (Amazon) | 完全無料（データと広告で回収） | 世界最大 |
| StoryGraph | フリーミアム＋Plus $4.99/月 | Goodreads代替として急成長 |
| Bookly | フリーミアム（無料10冊制限）＋$30/年 | 読書タイマー系 |

**弱点・不満**: 読書メーターは感想255文字制限・CSVエクスポート不可・本しか登録できない。ブクログは統計が貧弱。両方ともUIは古く、ソーシャル前提。Booklyは無料版が実質体験版。

**参入余地: ほぼ無い（単独カテゴリとしては）**。書誌DB・バーコードスキャン・レビュー資産の壁が高い。汎用アプリの1カテゴリとして「私的なメモ＋写真（サイン本、読んだ場所）」を拾う程度が現実的。

## 2. 映画

| アプリ | モデル | 規模感 |
|---|---|---|
| Filmarks（つみき） | 無料＋広告＋プレミアム月550円 | 月間利用者約592万人（2025年末）、累計2億Mark超。2023年の840万人から**減少傾向** |
| Letterboxd | 無料＋Pro $19/年＋Patron $49/年 | 2026年初に2,600万〜3,000万人と爆発的成長 |

**弱点・不満**: Filmarksプレミアムは「鑑賞記録のビジュアライズ」「広告非表示」を月550円のサブスク壁の内側に置く。Letterboxd Proの中核価値も「統計」。**つまり「自分の記録を美しく見る」に支払い意思があることを両者が証明済み**。

**参入余地: 単独では無理、だがヒントが濃い**。作品DBは持てないが、「統計・ビジュアライズは買い切りに全部入り」は明確なカウンター。ミニシアター・劇場体験（劇場、半券、同行者）の記録は Filmarks が弱い。

## 3. 美術館・博物館 — ✅ 手薄

| アプリ | モデル | 性格 |
|---|---|---|
| チラシミュージアム（イープラス） | 無料（チケット送客） | 展覧会**情報**アプリ。記録機能は薄い |
| Tokyo Art Beat | 無料＋月120円課金 | 情報＋簡易記録 |
| Punct Diary | 無料＋広告＋IAP | 個人開発の観覧記録専用（2024年〜） |
| puce | 無料 | 展覧会レビュー共有（ソーシャル型） |

**評価: 明確にアンダーサーブド**。「美術館 鑑賞記録 アプリ」の検索結果が個人開発アプリ・note の工夫記事・「Notion で代用」記事で埋まっている＝決定版が存在しない。大手（イープラス）は送客が本業で記録に投資する動機がない。展覧会は会期情報の DB 化が難点だが、**自分で入力する私的記録なら DB 不要**——Mystorium と同じ構造。

## 4. 演劇・観劇 — ✅ 手薄（新規参入は増加中）

| アプリ | モデル | 性格 |
|---|---|---|
| シアティ Theaty（カンフェティ系） | 無料（チケット送客・チェックインP） | 観劇記録＋推し活。「チェックイン不具合」「写真登録できない」等の品質不満レビューあり |
| CoRich舞台芸術 | 無料（口コミ＆予約ポータル） | 小劇場口コミ。UI古い |
| TheaterRecords / LiveNote / Live Rock | 個人開発 | 観劇・ライブ記録特化・小粒 |

**評価: アンダーサーブド**。知恵袋で「使いやすい観劇記録アプリは？」→回答が Notion や Google Keep という状況。観劇クラスタは**座席番号・マチソワ・キャスト変更・複数回観劇**という特有ニーズがあり、Mystorium のイマーシブ知見・リピートグループ機能と直接隣接。**開発者の既存ユーザー層との重なりが最も大きい**。

## 5. 食べ歩き・飲み歩き

- 食べログ＝検索インフラ（自分の記録ツールではない・営業電話やスコア不透明で悪評もある）。Retty・SARAH はコミュニティ型で事業は苦戦気味。「自分だけのグルメマップ」系は個人開発が乱立・小粒。
- **評価: 「私的記録」に限れば空いている**が、**Google マップの保存リスト＋写真という強力な無料代替**がいる。地図（MapKit）は Mystorium の既存資産が効く領域。

## 6. 汎用記録・コレクション — ⚠️ 直接競合あり

| アプリ | モデル | 性格 |
|---|---|---|
| **キログ**（個人開発・日本） | 無料＋**買い切り**課金、控えめ広告 | 記録項目カスタマイズ・統計・カレンダー。評判良好。**最も近い直接競合** |
| 記録｜読書も映画も | 無料 | シンプル系。メディア紹介あり |
| Sofa (iOS) | フリーミアム＋$29.99/年 | 美しいUI（MacStories絶賛）だが**サブスクへの不満レビューあり**。日本語・日本の趣味文脈なし |
| Day One | サブスク $34.99〜/年 | 13年ユーザーの「値上げ・複雑化」離反スレッドあり |
| Notion / Google Keep | 無料〜 | 「観劇記録は Notion 自作」勢が多い＝専用アプリへの潜在需要の証拠 |

**評価: 空白ではない**。キログが「なんでも記録・買い切り」を既に取っている。ただし機能表で戦うユーティリティ的な見た目で、「写真が主役・美しい統計・愛でる」路線とは別物。差別化軸は「機能の自由度」ではなく「体験の美しさ×日本の趣味文脈」。

## 7. 推し活・ライブ参戦記録 — ⚠️ 市場は熱いが競争激化

- 市場規模: 推し活人口約1,384万人・市場3.5兆〜4.1兆円（2025年）。年平均17%成長。
- Oshibana（推し活管理の定番・無料無広告=マネタイズ薄・記録統計は主戦場外）/ シアティ / LiveFans（セトリDBが本体）/ LIVE DIARY・LiveNote 等の個人開発参戦記録が2023〜25年に複数リリース＝**参入ラッシュ中**。
- **勝ち筋は単発機能ではなく「観劇・ライブ・展示をまたぐ統合記録」**（推しは複数ジャンルを横断する）＋チケット半券・座席の美しいアーカイブ。

## 繰り返し観測された攻めどころ（弱点パターン）

1. **統計・ビジュアライズの課金壁**: Filmarks・Letterboxd・StoryGraph・ブクログ全員が「自分の記録を美しく見る」を有料化 → 買い切りに全部入りは明確な対抗軸
2. **ソーシャル前提への疲れ**: 「自分だけの記録」需要は Notion 自作勢の存在が証明
3. **サブスク疲れ**: Day One 離反・Sofa への「買い切りにして」レビュー
4. **エクスポート不能・データ人質**: 読書メーターCSV不可など → エクスポート自由は信頼の売り物
5. **ニッチ記録アプリの品質不足**: シアティの不具合レビュー・個人開発の素朴なUI → デザイン品質で上回る余地大
