# FAVORECO 初回オンボーディング文言案

## 1. ようこそ

```text
ようこそ！
FAVORECOへ

ここは、あなたの「好きだった」を
きれいに残していく場所です。

観た作品。
行った場所。
心に残った時間。

FAVORECOは、そんな体験を
あとから何度でも思い出せる記録にする
お手伝いをします。

まずは少しだけ、
あなたのことを教えてください。
```

ボタン:

```text
はじめる
```

下部:

```text
続けることで、利用規約・プライバシーポリシーに同意したものとみなされます。
```

## 4. できること

```text
〇〇さんですね！

FAVORECOでは、あなたの体験を
こんなふうに残せます。
```

リスト:

```text
・観た日、行った場所、評価をまとめて記録
・写真やメモを、思い出として一緒に保存
・チケット、座席、金額、人物も残せる
・申込、当落、入金、発券、当日の予定までまとめて管理
・これから行く予定や、あとで記録したいものを一時保存
・ジャンルをまたいで、好きな体験を見返せる
```

締め:

```text
細かく残す日も、
ひとことだけ残す日も。

予定の管理から、終わったあとの思い出まで。
あなたのペースで使えます。
```

ボタン:

```text
わかった
```

## 5. 開始

```text
準備できました。

あなたの「好き」は、
ちゃんと残しておく価値があります。

何気ない一日も、
忘れたくない体験も、
あとから見返せる思い出にしていきましょう。

FAVORECOで、
最初の記録を始めますか？
```

ボタン:

```text
思い出を記録する
```

## 7. 追加メニュー

```text
今すぐ記録するジャンルを選びます。

まだ詳しく決まっていないものは、
「あとで記録」に入れておけます。
```

## 8. 基本情報

```text
まずは、思い出の表紙を作るところから。

タイトル、日付、場所、アイキャッチ画像。
これだけで、あなたの体験が
ひとつの記録として残ります。
```

## 9. 入力ユニット

```text
もっと残したいときは、
必要な項目だけ開いてください。

写真、人物、チケット、金額、メモ。
思い出の濃さに合わせて、
自由に足していけます。
```
