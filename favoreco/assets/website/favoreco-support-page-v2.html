SUPPORT

サポート

記録、予定・チケット、同期・バックアップなど、Favorecoを安心して使うための情報をご案内します。

Favorecoは現在、App Store公開に向けて開発中です

画面、機能、対応プラン、料金は公開までに変更される場合があります。公開時点の正確な内容はアプリ内とApp Storeの表示をご確認ください。更新情報は公式Xでお知らせします。

基本的な使い方

01 / START

記録したいジャンルを選ぶ

初回起動時に映画、観劇、ライブ、美術展、書籍、お酒、おでかけ施設、御朱印などから選択します。表示ジャンルは後から変更できます。自作ジャンルは対応プランで利用できます。

02 / ADD

中央の「＋」から記録する

「思い出を記録」ではジャンルを選んで詳しく登録し、「クイック記録」では気になる情報を一時保存できます。予定・チケットは記録またはカレンダー右上のチケット一覧から追加します。

03 / TICKET

予定とチケットを管理する

申込前、当落待ち、当選・落選、入金待ち、発券待ち、参加済みなどを管理できます。通知を有効にすると、設定した期限を端末のローカル通知でお知らせします。

04 / MEMORY

カレンダーと統計で振り返る

予定と記録を月表示で確認し、記録数、ジャンル傾向、月刊・年間Favorecoから過去の体験を振り返れます。一部機能は対応プランで利用できます。

データとバックアップ

通常は端末内へ保存されます

記録、写真、予定、チケット情報は、初期状態ではこのiPhoneのアプリ領域へ保存されます。アプリを削除すると端末内データも削除されるため、機種変更や修理の前にバックアップをご確認ください。

用途に合わせて書き出せます

  • JSON / CSV：確認や移行用の軽量な書き出しです。写真本体は含まれません。
  • 写真付き完全バックアップ：記録と写真を含む.favorecobackupファイルです。復元時は内容を確認してから既存データへ統合します。
  • iCloud同期・自動バックアップ：対応プランで利用者が有効にした場合のみ、CloudKitまたはiCloud Driveを利用します。

大切な記録は複数の方法で保全してください。

iCloud同期はバックアップそのものと同じではありません。端末故障や誤操作に備え、定期的な写真付き完全バックアップもおすすめします。

写真・通知・カレンダー

写真とカメラ

写真は利用者が選択したもの、またはカメラで撮影したものだけを記録へ追加します。保存時には長辺を縮小して再描画し、元画像の位置情報などのメタデータを原則として引き継ぎません。

通知

通知全体を有効にした時だけiOSの通知許可を求めます。申込開始・締切、当落、入金、発券、公演日、会員期限、月刊・年間レポートは、アプリ内設定から種類ごとに変更できます。

カレンダー

許可した場合、iOSカレンダーの予定をFavoreco内へ読み取り専用で重ねて表示できます。また、利用者が明示的に操作した予定だけを、選択したカレンダーへ追加・更新・削除できます。自動で無関係な予定を書き換えることはありません。

購入とプラン

アプリ内購入とサブスクリプションはAppleのStoreKitを通じて処理されます。購入履歴の確認、復元、サブスクリプションの管理はApple Accountに基づきます。決済情報はRANOVIQOへ直接送信されません。

App Store公開前のため、現在Webサイトに掲載している機能区分や料金は予定です。公開時はApp Storeおよびアプリ内の購入画面に表示される内容が優先されます。

よくある質問

記録したデータはどこに保存されますか？

初期状態では端末内です。対応プランで利用者がiCloud同期または自動バックアップを有効にした場合は、AppleのCloudKitまたはiCloud Driveにも保存されます。

機種変更前に何をすればよいですか？

iCloud同期の完了状態を確認するか、写真付き完全バックアップを書き出してください。JSON / CSVには写真本体が含まれないため、写真も移行する場合は完全バックアップをご利用ください。

「データを安全に開けませんでした」と表示されます。

データ保護のため通常画面を停止しています。アプリを終了してiPhoneを再起動し、空き容量とiCloud状態をご確認ください。アプリの削除や再インストールは、バックアップの有無を確認するまで行わないでください。

誤って記録やジャンルを非表示にしました。

非表示は削除ではありません。設定の「データ管理」から非表示対象を確認し、再表示できます。完全削除を実行したデータは、バックアップがない限り復元できません。

写真から位置情報などが送信されますか？

記録用画像は端末内で再描画・圧縮し、元画像の位置情報などのメタデータを原則として引き継ぎません。OCRもApple Visionを使い、原則として端末上で処理します。

位置情報の許可は必要ですか？

現在の実装では端末の現在地権限を必須としていません。Apple Mapsの候補検索から選んだ住所・座標、または利用者が入力した場所を記録へ保存できます。

通知が届きません。

iPhoneの「設定」→「通知」→「Favoreco」と、アプリ内の「設定」→「通知」の両方をご確認ください。過去日時の通知や、必要な日付が未入力の通知は予約されません。

チケットサイトのパスワードは保存されますか？

パスワード本体は通常の記録データやCloudKitへ保存しません。ログインID、会員番号、有効期限などを任意で記録できますが、パスワードや決済情報はメモ欄にも入力しないでください。

Appleカレンダーの予定は変更されますか？

外部予定の重ね表示は読み取り専用です。Favorecoの予定をカレンダーへ反映する機能では、利用者が選んだ予定とカレンダーだけを明示操作により追加・更新・削除します。

アプリを削除するとデータも消えますか？

端末内データは削除されます。iCloud上の同期データやFilesへ書き出したバックアップは別に残る場合があります。再インストール前に、利用中の保存先とバックアップをご確認ください。

退会やアカウント削除は必要ですか？

現在、Favoreco独自のユーザーアカウントはありません。アプリ内の「全データ削除」で記録・写真・予定等を削除できます。問い合わせメールなど運営者が保有する情報の確認・削除は、下記窓口へご連絡ください。

チケットの購入・当選・入場は保証されますか？

Favorecoは予定と申込状況を利用者自身で整理するための記録アプリです。各プレイガイドや主催者の公式サービスではなく、購入、抽選結果、発券、入場を保証しません。最終確認は必ず公式案内をご覧ください。

お問い合わせの前に

可能な範囲で、iPhoneの機種名、iOSとFavorecoのバージョン、問題が発生した画面、再現手順、表示されたエラーをお知らせください。

送らないでください：パスワード、認証コード、決済情報、チケット番号、会員番号、バックアップファイル、第三者の個人情報。

スクリーンショットを添付する場合は、氏名、座席、アカウント情報などを隠してください。バックアップや写真が調査に必要な場合も、こちらから個別にお願いするまでは送信しないでください。
